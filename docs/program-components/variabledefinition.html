Variable Definition

Here is an example of a Xenon Variable Definition

int32 i = 0;

This Variable Definition is broken down into the following tokens.

Keyword: int32
Identifier: i
Equals
IntegerLiteral: 0
Semicolon

In this scenario, you passed a datatype of int32. The compiler will reserve the required space for the data type. You can also pass a datatype that can be implicitly converted.
